### Отчет по лабораторной работе №2: "Основы обработки цифровых изображений"

#### **1. Цель работы**

Целью работы являлась практическая реализация и исследование алгоритмов обработки цифровых изображений. Согласно варианту №10, были выбраны следующие методы:
1.  Глобальная пороговая обработка: реализованы ручной метод и автоматический метод Оцу.
2.  Нелинейные фильтры на основе порядковых статистик: реализованы фильтры минимума, максимума и медианный фильтр.

#### **2. Реализация**

Для выполнения работы было создано приложение на Python с использованием `tkinter` для интерфейса. Библиотека `numpy` была задействована для ускорения вычислений в методе Оцу.

Интерфейс приложения разделен на две части: слева отображается оригинальное изображение, а справа — результат обработки. Панель управления позволяет:
*   Загружать изображение.
*   Применять выбранные фильтры и методы.
*   Регулировать параметры (размер ядра для фильтров, порог для ручной бинаризации).
*   Сбрасывать изменения и сохранять результат.

#### **3. Описание реализованных методов**

##### **3.1. Глобальная пороговая обработка**

Глобальная пороговая обработка (бинаризация) — это процесс сегментации изображения на два класса пикселей (обычно фон и объект) на основе порогового значения яркости `T`.

`f(x, y) = { 255, если I(x, y) > T;  0, если I(x, y) <= T }`

*   Ручная пороговая обработка: В приложении реализован ползунок, позволяющий пользователю вручную задавать пороговое значение `T` в диапазоне от 0 до 255. Этот метод прост, но его эффективность сильно зависит от конкретного изображения и условий освещения. Он требует ручного подбора оптимального значения для каждого случая.

*   Метод Оцу: Это алгоритм автоматического определения оптимального порога `T`. Его суть заключается в том, чтобы найти такое пороговое значение, которое максимизирует межклассовую дисперсию. Иными словами, алгоритм стремится разделить гистограмму яркостей на две группы так, чтобы разброс яркостей *внутри* каждой группы был минимален, а разброс *между* группами — максимален. Это делает классы (фон и объект) максимально различимыми. Метод Оцу эффективен для изображений с бимодальной гистограммой (когда есть два явных пика яркости).

##### **3.2. Нелинейные фильтры на основе порядковых статистик**

Эти фильтры работают по принципу скользящего окна (ядра, апертуры). Для каждого пикселя изображения анализируются его соседи в пределах окна (например, 3x3 или 5x5). Значение центрального пикселя заменяется результатом, вычисленным на основе упорядоченного (отсортированного) списка значений всех пикселей в окне.

*   Фильтр минимума (Min Filter): Значение центрального пикселя заменяется на минимальное значение среди всех пикселей в окне. Этот фильтр приводит к затемнению изображения, увеличению темных областей и уменьшению светлых объектов.

*   Фильтр максимума (Max Filter): Значение центрального пикселя заменяется на максимальное значение в окне. Эффект противоположен фильтру минимума: изображение становится светлее, светлые области расширяются.

*   Медианный фильтр (Median Filter): Значение центрального пикселя заменяется на медиану (центральный элемент в отсортированном списке) значений всех пикселей в окне. Главное преимущество этого фильтра — эффективное удаление импульсного шума ("соль и перец"), поскольку случайные яркие или темные пиксели-выбросы оказываются по краям отсортированного списка и не влияют на медиану. При этом медианный фильтр сохраняет резкость границ объектов значительно лучше, чем линейные сглаживающие фильтры (например, размытие по Гауссу).

Созданное приложение позволило наглядно сравнить ручной и автоматический (Оцу) подходы к бинаризации, а также изучить эффекты от применения фильтров минимума, максимума и медианы для задач подавления шума и изменения геометрии объектов на изображении.
